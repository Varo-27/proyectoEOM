# Semantic Explorer — Frontend

Interfaz React del explorador semántico.

## Desarrollo

Con Docker Compose (recomendado), el frontend se sirve en el puerto **8080**.

Para desarrollo directo con Vite:

```bash
cd frontend
pnpm install
pnpm dev
```

Variables en `frontend/.env` (ver `.env.example` en la raíz del monorepo).
